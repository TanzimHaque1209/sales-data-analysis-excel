Sales Data Analysis using Excel

 Overview

This project presents a sales data analysis performed using Microsoft Excel. The objective was to explore and analyze sales performance across territories, brands, and categories, and to generate insights using data visualization and forecasting techniques.

---

 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Slicers
* Data Visualization (Bar, Doughnut, Line Charts)
* Forecast Sheet (Time Series Forecasting)

---

Key Analysis Performed

* Territory-wise delivered amount analysis
* Category-wise and brand-wise performance evaluation
* Order vs Delivered amount comparison
* Bounced value calculation (Order - Delivered)
* Area-wise visualization using doughnut charts
* Date-wise trend analysis
* Forecasting delivered amount for January 2025
